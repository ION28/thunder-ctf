Run in cloud shell or on a local machine with the following prerequisites:
- Make sure gcloud cli is installed
- python3 installed
- PATH of shell that is executing python must include git
mkdir t-ctf
cd t-ctf
git clone https://github.com/NicholasSpringer/thunder-ctf.git
virtualenv -v thunder-env
source thunder-env/bin/activate
cd thunder-ctf
pip3 install -r requirements.txt 
  
You will need to make a new google cloud project to run the ctf on. 
Enable billing on the project.
gcloud auth login
gcloud config set project [PROJECT-ID]
gcloud auth application-default login
python3 thunder.py set_project [PROJECT-ID]
to deploy level instances:
  python3 thunder.py create [level-name]
to destroy level instances:
  python3 thunder.py destroy [level-name]
full list of commands:
  python3 thunder.py help
